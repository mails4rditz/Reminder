pdf.js (v3.9.179) and pdf.worker.js are included in this folder.

The extension uses these builds for PDF parsing out-of-the-box. If you want a newer version, replace these files with the builds from `pdfjs-dist` (see the `README_EXTENSION_SETUP.md` for instructions).
